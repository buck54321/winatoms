Install

To play offline, you can use the Python scripts available here. There are three scripts,
create.py
,
play.py
, and
verify.py
. Like this website, the scripts rely on the Decred Python Toolkit, which can be installed with
pip install decred
.

Create

You can create a game with
python create.py
or
python create.py --testnet
. The script will generate the game key and the contract address. Do not lose the game key or forget your answer. Both are required for redemption.

Play

The player must have a Decred address. Create an address using Decrediton or dcrwallet. Or, if you're feeling adventurous, try out TinyWallet, an an in-development system tray wallet written in Python.
To play, run
python play.py
. You will be prompted for the game key and your redemption address. Then guess away.
The challenge of offline play is managing distribution of the game key or otherwise allowing the user to get the transaction signature(s) needed to redeem. The game key cannot be publicized, since a malicious node or miner in posession of the game key could attempt to steal the funds when the redemption is broadcast.

Verify

When you create a challenge on Win Atoms!, you should independently verify the game key with the verification script.
python verify.py <paste-game-key-here>
. You will be prompted for your solution, which will be validated against the game key. You'll then receive a failure or success message, along with some basic data about the game key.
